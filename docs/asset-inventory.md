## Verification Steps

After adding assets:

1. Confirm file casing matches the paths above exactly.
2. Run the app locally.
3. Open every section and check the browser console for unintended image 404s.
4. Review card crops at desktop and mobile widths.
5. Review modal image sizing at desktop and mobile widths.
6. Update this inventory and `docs/tasks.md`.
